# CDD Workshop - Mac Setup Scripts

## Overview

These scripts help participants prepare their Mac for the Context-Driven Development workshop.

## Files

### `setup-mac.sh`
Automated installation script that sets up the complete development environment.

**Installs:**
- Xcode Command Line Tools
- Homebrew (package manager)
- Node.js v20 (includes npm)
- Python 3.12
- Git

**Usage:**
```bash
curl -fsSL https://raw.githubusercontent.com/itayshmool/creators-training-2026/main/scripts/setup-mac.sh | bash
```

### `validate-setup.sh`
Quick validation script to verify all tools are properly installed.

**Usage:**
```bash
curl -fsSL https://raw.githubusercontent.com/itayshmool/creators-training-2026/main/scripts/validate-setup.sh | bash
```

## Requirements

- macOS 11 (Big Sur) or newer
- Internet connection
- ~5GB free disk space

## Setup Time

- First-time setup: 10-15 minutes
- Validation: 30 seconds

## Support

If you encounter issues during setup, please screenshot the error and contact the workshop organizers.
